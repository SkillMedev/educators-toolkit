# Educator's Toolkit

**Plan the term, run the lesson, and grade it fairly - guidance for classroom teachers.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

For classroom teachers who want to spend less time planning and more time teaching. Reach for this pack when you are mapping a term, designing a lesson, adapting it for a mixed-readiness room, and turning student work into fair grades and feedback that actually moves learners forward - plus support for building study plans and synthesizing source material. The guidance is opinionated and pedagogy-first, not generic templates, so each step starts from real structure instead of a blank page.

- **From the catalog:** [skillme.dev/pack/educators-toolkit](https://skillme.dev/pack/educators-toolkit) — install the whole pack into Claude in one step.

## Skills in this pack

- **[Curriculum Mapper](skills/curriculum-mapper/SKILL.md)** — Maps a course's scope and sequence across a term or year - units, weeks, standards coverage with introduced/developed/mastered notation, and Bloom's-level progression - and flags gaps and redundancy.
- **[Lesson Plan Builder](skills/lesson-plan-builder/SKILL.md)** — Builds a complete single-lesson plan - measurable objective, timed four-part arc (hook, direct instruction, guided practice, closure), a formative check with a proceed/re-teach rule, and differentiation notes - clear enough that a substitute could run it.
- **[Differentiated Instruction](skills/differentiated-instruction/SKILL.md)** — Adapts an existing lesson or unit for mixed readiness levels using tiered assignments, flexible grouping, and scaffolds that fade - without lowering the learning objective.
- **[Quiz Generator](skills/quiz-generator/SKILL.md)** — Writes quizzes and assessments with a stated Bloom's-level distribution, misconception-based distractors, and an answer key with rationales - plus a summary table of level, type, and points per question.
- **[Rubric Builder](skills/rubric-builder/SKILL.md)** — Builds grading rubrics - analytic, holistic, or single-point - with criteria traced to learning objectives, 3-5 performance levels, observable behavior in every cell, and defensible weighting.
- **[Student Feedback Writer](skills/student-feedback-writer/SKILL.md)** — Writes specific, growth-oriented feedback on student work using the Glow-Grow-Go structure - evidence from the actual work, one improvement priority, and a concrete next action - sized to the context from draft comments to progress reports.
